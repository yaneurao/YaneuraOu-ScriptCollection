# YaneuraOu-Builder

`YaneuraOu-Builder` は、やねうら王のリリース用ビルドscriptを生成・実行するための
Python GUIです。GUIでビルド設定を編集し、具体的な build plan と実行scriptを
`runs/` 以下に出力します。
