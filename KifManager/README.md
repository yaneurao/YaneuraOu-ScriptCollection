# KifManager

棋譜のダウンロードと抽出を扱う補助ツールを置くフォルダです。

## scripts/

`scripts/` には、floodgate、WCSC、電竜戦、その他の棋譜から USI `position` 形式の棋譜行を抽出するCLIスクリプトと、棋譜ダウンロード用スクリプトを置いています。

## kif-manager.py

`kif-manager.py` は、棋譜抽出と棋譜ダウンロードを扱う Tkinter GUI です。

```bash
python3 kif-manager.py
```

上位タブは `棋譜抽出` と `棋譜のダウンロード` に分かれています。

`棋譜抽出` には、floodgate、WCSC、電竜戦、その他の抽出タブがあります。各タブで入力フォルダと出力ファイルを指定し、必要ならプレイヤー名フィルタを指定して `抽出` を押します。出力ファイルは `think_sfens.txt` がデフォルトです。floodgate の入力フォルダは `downloaded-kif/floodgate`、WCSC の入力フォルダは `downloaded-kif/wcsc` がデフォルトです。

floodgate では `開始日` / `終了日` を指定して、2026/06/01 から 2026/06/07 までのように対局日で絞り込めます。

WCSC では `開始年` / `終了年` を指定して、2020年から2025年までのような範囲抽出ができます。WCSCの `wcso1` は WCSC30 扱いで、2020年として扱います。

WCSCと電竜戦では `決勝に出場したソフトの棋譜のみ抽出` を有効にできます。WCSCでは同じ大会の決勝棋譜から、電竜戦では同じ本戦イベントの決勝リーグまたはA級棋譜からプレイヤー名を集め、そのどちらかが登場する予選/二次予選/決勝棋譜だけを抽出します。

`抽出対象のファイルをログに出力` を有効にすると、抽出対象として選択された棋譜ファイルのパスをログに出力します。デフォルトはオフです。

`棋譜のダウンロード` には、floodgate、WCSC、電竜戦のダウンロードタブがあります。

ダウンロード中は `ダウンロード` ボタンが `停止` に変わります。押すと停止要求を出し、現在処理中の1ファイルまたは1ページが終わったところで停止します。

floodgateタブでは `2026` のように対局年を指定します。ダウンロードした年別アーカイブは、デフォルトでは `downloaded-kif/floodgate/wdoor2025.7z` のように保存します。今年のものは前日までの棋譜がダウンロードされるため、`downloaded-kif/floodgate/wdoor2026-20260607.7z` のように本日の日付を付けて保存します。

WCSCタブでは `WCSC36` のように大会名を指定します。デフォルトでは `downloaded-kif/wcsc/wcsc36/` のように、出力フォルダ配下の大会名フォルダへ保存します。`live中継ページからダウンロード` はデフォルトでオフです。オフの場合は `http://www2.computer-shogi.org/kifu/kifu.html` の公式棋譜アーカイブから `.zip` または `.lzh` をダウンロードして展開します。オンにした場合だけ、WCSC24以前の `live2` 形式などのlive中継ページから個別棋譜を取得します。WCSC16以前のLZH展開には `lhafile` が必要です。`kifu6.lzh` のように `lhafile` だけでは展開できない古い圧縮形式は、7-Zip、bsdtar、unar のいずれかを外部コマンドとして使用します。WCSC30は中止になりましたが、オンライン大会として開催された棋譜は `WCSO1` と指定すれば `downloaded-kif/wcsc/wcso1/` にダウンロードできます。サーバー負荷を避けるため、アクセス間隔のデフォルトは 10 秒で、通常は 2 秒以上を指定します。

電竜戦タブでは、公式リンク集に掲載されている大会をコンボボックスから選択できます。`一覧更新` を押すと、公式リンク集から最新の大会一覧を取得します。デフォルトでは `downloaded-kif/denryu/dr6_production/` のように、出力フォルダ配下の大会キーごとのフォルダへ保存します。`live中継ページからダウンロード` はデフォルトでオフです。オフの場合は大会ページまたは公式リンク集から一括ZIPを探して展開します。一括ZIPが見つからない大会では、live中継ページの `kifulist.txt` から `kifufiles/*.csa` を取得する方式にフォールバックします。大会終了前で一括ZIPがまだ公開されていない場合も、このlive取得を使います。アクセス間隔のデフォルトは 10 秒で、通常は 2 秒以上を指定します。

出力は 1 対局 1 行です。

```text
startpos moves 7g7f 3c3d ...
```

同一の `startpos moves ...` 行は初出だけを出力します。

### プレイヤー名フィルタ

`both-player-list` は、先手と後手の両方がリスト内の正規表現に一致する棋譜だけを通します。

`either-player-list` は、先手または後手の少なくとも片方がリスト内の正規表現に一致する棋譜だけを通します。

両方を指定した場合は OR 条件です。どちらかの条件を満たす棋譜が抽出されます。

### 逆転棋譜フィルタ

`逆転棋譜` を有効にすると、先に年・プレイヤー名・rating・決勝出場ソフトなどの条件で絞り込んだ棋譜のうち、さらに評価値コメントから逆転が確認できる棋譜だけを出力します。

片方のプレイヤー自身が出力した評価値が一度指定値以上、または指定値以下になり、その後に同じプレイヤーの評価値が0をまたいだ場合に逆転棋譜として扱います。評価値コメントがない棋譜は、この条件を有効にした場合は除外されます。

### 設定保存

GUI の入力値は `kif-manager-settings.pickle` に保存され、次回起動時に復元されます。この設定ファイルはローカル設定なので git 管理対象外です。
